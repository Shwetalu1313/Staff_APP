Data collected

This static site does not include a database, user account system, or form submission backend.

  • No profile information is submitted from this site to a server controlled by this repo.
  • No payment, HR record, or ticket record is stored by this public static site itself.
  • Displayed names, counts, and ticket details are demo-style interface content.

Browser storage

The site stores a small theme preference in your browser.

  • The light or dark mode choice is saved in local storage so the same theme loads on the next visit.
  • If browser storage is blocked, the site still works but the theme preference may not persist.

External destinations

Some app links open external services.

  • When you open Google, Atlassian, or similar external destinations, those providers may collect data under their own policies.
  • You should review third-party privacy notices separately if those services are used.

Updates

This page describes the current static implementation of the site.

  • If tracking, analytics, forms, or authentication are added later, this privacy page should be updated to match.
  • The Version page records visible site changes so users can see when documentation or features were expanded.
