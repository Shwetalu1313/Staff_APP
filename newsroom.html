5G core upgrade completed

Mandalay region is now operating on the new core stack, giving the network team a more stable base for capacity expansion, faster fault isolation, and cleaner service migration planning.

Enterprise subscriptions increased by 11% this quarter, led by stronger uptake from mid-market accounts and better conversion from bundled service offers.

Commercial and support teams are using the result as an input for next-quarter retention planning, onboarding improvements, and service-priority alignment.

Digital onboarding rollout

All branches have now moved to paperless onboarding, reducing manual document handling and making status visibility easier for frontline teams and back-office reviewers.

The next focus is process tuning: shorten the longest approval steps, improve handoff tracking, and standardize branch feedback into one implementation loop.
