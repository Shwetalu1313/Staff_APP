v1.5.0

Navigation Documentation
  • Added global `Alt` keyboard shortcuts for core public pages and moved Technical Document access to `Alt + E`.
  • Created a dedicated Documentation page for system behavior, backend planning, palette guidance, searchable topic navigation, and layout customization notes.
  • Exposed Documentation from the shared footer, combined the former Services and Support flows into one Services page, added a dedicated AI page, and introduced per-page section hide/sort controls.

Footer Update Trust Pages
