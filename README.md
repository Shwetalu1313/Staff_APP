# ATOM Staff Portal

Static staff portal built with plain HTML, CSS, and JavaScript.

Live site:

`https://shwetalu1313.github.io/Staff_APP/`

Current public pages:

- Home
- Workspace
- Services
- Apps
- AI Assistant
- FAQ
- Documentation
- Settings
- Version
- Legal
- Privacy

Current frontend features:

- Shared top navigation with active page state
- Light, dark, and system theme modes
- Dynamic weather widget in the header
- Desk pets with local preference storage
- Atom Voice anonymous room demo
- Dedicated AI Assistant page
- Per-page section hide and drag-sort controls
- Searchable developer documentation page

Tech stack:

- `index.html` and additional static HTML pages
- `styles.css` for shared layout, theme, and component styling
- `main.js` for shared UI behavior and browser-local state

Notes:

- This is currently a static frontend project.
- User settings, layout preferences, and demo chat data are stored in browser local storage.
- Some features are demo-only until a backend service is connected.
